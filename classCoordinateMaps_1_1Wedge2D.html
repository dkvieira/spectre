Detailed Description

Two dimensional map from the unit square to a wedge, which is constructed by interpolating between a circular arc of radius radius_of_circle and a flat face which is circumscribed by a circular arc of radius radius_of_other. These arcs extend \(\pi/2\) in angle, and can be oriented along the +/- x or y axis. The choice of using either equiangular or equidistant coordinates along the arcs is specifiable with with_equiangular_map. For a more detailed discussion, see the documentation for Wedge3D.
